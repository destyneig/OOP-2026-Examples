# Chapter 3 - Arrays

Code examples for Chapter 3 (Arrays) from *Learning JavaScript Data Structures and Algorithms (4th Edition)*.

## Use as a VS Code project

1. Open this folder in VS Code: `week8/Chapter03-Arrays`
2. Run the menu:
   - `npm start` (interactive)
   - `node main-menu.js --list` (list examples)
   - `node main-menu.js <number>` (run a specific example)
3. (Optional) Use the VS Code launch config: **Chapter 3 Main Menu**.
